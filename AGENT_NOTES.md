# AGENT_NOTES.md — Loslasszen Website

Project-specific operating notes for Sage and any delegated coding/design work on the Loslasszen website.

## Release / Git Safety

- Never push to `main` without Philipp's explicit request.
- Never deploy, publish, or otherwise ship changes without Philipp's explicit request.
- Default stopping point is one of:
  - local uncommitted changes
  - a local commit
  - a clear summary of what changed and what is ready
- If the instruction is ambiguous, ask before pushing.

## Design Direction

- Preserve the calm, minimal, elegant Loslasszen aesthetic unless Philipp explicitly asks for a different direction.
- Prefer simple, visually coherent solutions over feature bloat.
