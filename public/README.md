# Public Assets Folder

This folder contains static assets that will be served directly by your website.

## Open Graph Image (og-image.jpg)

**Place your og-image.jpg file directly in this `public` folder.**

### Requirements:
- **Filename:** `og-image.jpg` (exactly this name)
- **Dimensions:** 1200 x 630 pixels (recommended for optimal display across all social platforms)
- **Format:** JPG or PNG (JPG recommended for smaller file size)
- **File size:** Keep under 1MB for fast loading
- **Content suggestions:**
  - A screenshot of your beautiful landing page
  - A photo of you working/treating clients
  - Your logo with a branded background
  - A composition featuring your sumi-e cherry blossom theme
  - Text overlay with your tagline: "loslasszen Eberswalde | Endlich loslassen"

### Why this matters:
✅ Increases click-through rate from social shares by ~40%  
✅ Makes your links look professional on Facebook, Twitter, LinkedIn, WhatsApp  
✅ Controls how your site appears when shared  

### How to create your OG image:
1. Use Canva (free): Create a 1200x630px design
2. Take a screenshot of your website and crop to 1200x630px
3. Use Figma or Photoshop to design a branded image
4. Use an OG image generator tool online

### File location:
```
public/
  ├── og-image.jpg  ← Place your file here
  └── README.md     ← You're reading this now
```

The file will be automatically accessible at: `https://yourdomain.com/og-image.jpg`
